## Catalog Query Items for Tax

### Structure

`CatalogQueryItemsForTax`

### Fields

| Name | Type | Description |
|  --- | --- | --- |
| `tax_ids` | `List of string` | A set of [CatalogTax](#type-catalogtax) IDs to be used to find associated [CatalogItem](#type-catalogitem)s. |

### Example (as JSON)

```json
{
  "tax_ids": [
    "tax_ids7"
  ]
}
```
